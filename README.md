# Powerbi_dashboard_project
This project presents an interactive Super Store Sales Dashboard built in Microsoft Power BI to analyze sales performance, profit, orders, shipping, customer segments, and sales forecasting. The dashboard helps users monitor business performance through visually appealing and interactive reports.

**Dashboard 1 – Super Store Sales Dashboard**
The first dashboard provides an overall business performance analysis with the following features:

KPI Cards
Total Sales: 1.6M
Total Orders: 22K
Total Profit: 175K
Average Shipping Days: 4
Sales Analysis
Sales by Region (Donut Chart)
Sales by Customer Segment
Sales by Payment Mode
Monthly Sales Trend (2019 vs 2020)
Monthly Profit Trend (2019 vs 2020)
Sales by Ship Mode
Sales by Product Category
Top Sub-Categories
State-wise Sales & Profit Map
Interactive Filters
Region slicers (Central, East, South, West) for dynamic analysis.

**Dashboard 2 – 15-Day Sales Forecast**
The second dashboard focuses on sales forecasting using Power BI forecasting features.
Features:
Historical Sales Trend
15-Day Sales Forecast
Forecast Confidence Interval
State-wise Sales Comparison
Interactive Zoom Slider for detailed analysis

# This dashboard helps predict future sales trends, enabling better inventory planning and business decision-making.
